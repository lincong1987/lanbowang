插件方法详细
{Object} ligerGrid(p)
描述:
  • 1,支持本地数据和服务器数据(配置data或者url)
  • 2,支持排序和分页(包括Javascript排序和分页)
  • 3,支持列的“显示/隐藏”
  • 4,支持明细行(表格内嵌)
  • 5,支持汇总行
  • 6,支持单元格模板
  • 7,支持编辑表格(ligerGrid的一个特色,需要其他表单插件的支持)
  • 8,支持树表格(待加入)
  • 8,支持分组(待加入)

参数列表:
参数名 类型 描述 默认值
p {Object} 主要参数
p.width {String|Int} 宽度值,支持百分比 'auto'
p.height {String|Int} 高度值,支持百分比 'auto'
p.columnWidth {Int} 默认列宽度 120
p.resizable {String} table是否可伸缩(暂不支持) false
p.url {String} ajax url false
p.usePager {String} 是否分页 true
p.page {Int} 默认当前页 1
p.total {Int} 总页面数 1
p.pageSize {Int} 每页默认的结果数 10
p.pageSizeOptions {Array} 可选择设定的每页结果数 [10, 20, 30, 40, 50]
p.columns {Array} 数据源 []
p.columns[i].name {String} 表格列名
p.columns[i].totalSummary {Object} 汇总
p.columns[i].totalSummary.align {String} 汇总对齐方式
p.columns[i].totalSummary.type {String} 汇总类型sum,max,min,avg ,count
p.columns[i].totalSummary.render {Function} 汇总渲染器，返回html加载到单元格

p.renderDate {Function} 获取时间默认函数,可重载 function (value)
{
var da;
if (!value) return null;
if (typeof value == 'object')
{
return value;
}
if (value.indexOf('Date') > -1)
{
da = eval('new ' + value.replace('/', '', 'g').replace('/', '', 'g'));
}
else
{
da = eval('new Date("' + value + '");');
}
return da;
}
