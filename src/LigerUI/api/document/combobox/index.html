插件方法详细
{Object} ligerComboBox(p)
描述:
  • 1,可以将一个Select的表单对象转换成自定义下拉框。
  • 2,可以将一个type=text的表单对象转换成自定义下拉框(需要配置数据源)。
参数列表:
参数名 类型 描述 默认值
p {Object} 主要参数
p.data {Array} 数据源.JSON格式 null
p.tree {Object} 下拉框以树的形式显示，tree的参数跟ligerTree的参数一致 null
p.grid {Object} 下拉框以Grid的形式显示，grid的参数跟ligerGrid的参数一致 null
p.resize {Bool} 是否调整大小 true
p.slide {Bool} 是否以动画的形式显示 true
p.isMultiSelect {Bool} 是否多选 false
p.isShowCheckBox {Bool} 是否选择复选框 false
p.selectBoxWidth {Bool} 下拉框的宽度 false
p.selectBoxHeight {Bool} 下拉框的高度 false
p.textField {String} text对应的字段名 'text'
p.valueField {String} value对应的字段名 'id'
p.valueFieldID {String} 隐藏域的控件ID，如果页面不存在，自动创建一个 null
p.initValue {String} 初始化Value null
p.initText {String} 初始化Text null
p.split {String} 分割符号,多选时有效 ";"
p.columns {Array} 将下拉框转换为表格的模式 null
p.columns[i].name {String} 表格列名
p.columns[i].header {String} 表格列标题
p.columns[i].width {String} 表格列宽度
p.onBeforeSelect {Function} 选择前事件,可以通过返回false来阻止选择
  • 参数1：value
  • 参数2：text
 false
p.onSelected {Function} 选择后事件
  • 参数1：value
  • 参数2：text
 null
p.onStartResize {Function} 下拉框开始调整大小事件 null
p.onEndResize {Function} 下拉框停止调整大小事件 null
p.hideOnLoseFocus {Bool} 失去焦点时是否隐藏 true
p.url {String} 数据源URL(需返回JSON) null
p.onSuccess {Function} Ajax读取成功事件 null
p.onError {Function} Ajax读取失败事件 null
